# Tetris

Tetris is a block stacker game. The object is to clear as many lines as possible while surviving the relentlessly increasing gravity.
Note that this game uses custom keybinds. For more information, see below.

Basic Controls:
B - Left
A - Right
D - Soft Drop (temporarily increase gravity speed)

Advanced Controls:
S - Pause
H - Rotate 180
I - Left Rotate
M - Right Rotate
O - Hard Drop (instant drop)

Any pull requests or contributions are welcome.
